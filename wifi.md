# wifi

# 事前準備

nmcliコマンドおよびnmtuiコマンドが見つからない場合は、NetworkManagerを以下のコマンドでインストールする。

```
sudo apt-get install network-manager
```

## Wi-Fiをオンにする

```
nmcli radio wifi on
```

## nmtuiを起動する

```
nmtui
```

「接続をアクティベートする」を選択する

「終了」を選択する
