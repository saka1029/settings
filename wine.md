# wine

## 日本語フォントのインストール

一度```winecfg```を起動する。

dataにある以下のファイルを

* msgothic.ttc
* msmincho.ttc

以下のディレクトリにコピーする。

```
~/.wine/drive_c/windows/Fonts
```

## Paint Shop Pro 9インストール

ファイルサーバーから~/pspにインストールイメージをコピーする。

以下のコマンドでインストーラを起動する。

```
wine psp/Paint\ Shop\ Pro\ 9/autorun.exe
```

旧バージョンのインストールディレクトリを聞かれたら以下を入力する。

```
Z:\home\saka1029\psp\Paint Shop Pro 7
```
